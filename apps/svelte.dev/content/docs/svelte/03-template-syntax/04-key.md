```svelte
<!--- copy: false  --->
{#key expression}...{/key}
```

key ブロックは、式の値が変化したときにその内容を破棄して再生成します。コンポーネントを囲んで使用した場合、コンポーネントの再インスタンス化され、再初期化されます:

```svelte
{#key value}
	<Component />
{/key}
```

値が変更されるたびにトランジションを再生したい場合にも便利です:

```svelte
{#key value}
	<div transition:fade>{value}</div>
{/key}
```
