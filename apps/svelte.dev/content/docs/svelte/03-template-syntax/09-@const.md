title: {@const ...}

`{@const ...}` タグはローカル定数を定義します。

```svelte
{#each boxes as box}
	{@const area = box.width * box.height}
	{box.width} * {box.height} = {area}
{/each}
```

`{@const}` は、`{#if ...}`、`{#each ...}`、`{#snippet ...}` などのブロックや、`<Component />`、または `<svelte:boundary` の直下の子としてのみ使用可能です。
